E-Voting System – A Secure Digital Voting Platform

The E-Voting System is a highly secure and efficient online voting platform built using ReactJS for the frontend and Java Spring Boot for the backend. This system ensures integrity, security, and transparency in digital elections by leveraging modern encryption techniques.

Key Features

  • User Authentication – Secure login system to verify voters' identities.
  • End-to-End Encryption – Uses SHA (Secure Hash Algorithm) to ensure data integrity.
  • Tamper-Proof Voting – Votes are securely stored and cannot be altered.
  • Real-Time Vote Counting – Instantly calculates and displays election results.
  • Role-Based Access – Separate dashboards for admin, candidates, and voters.
  • Responsive UI – A modern, user-friendly interface using ReactJS and Bootstrap.
